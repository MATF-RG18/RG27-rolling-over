# RG27-rolling-over
Rolling over

Strelicama pomeramo lopticu levo ili desno u cilju da izbegnemo roze prepreke i da prikupimo sto vise zutih dijamanata. Posle odredjenog vremena put ce poceti da se okrece, dok se ne prevrne naopako, pa ce to dodatno otezati igricu.
